Este repositorio es una colección de imágenes generadas mediante inteligencia artificial,
donde exploro la combinación entre creatividad humana y tecnología avanzada para la creación de arte digital.

Aquí experimento con diferentes herramientas de IA generativa, utilizando prompts cuidadosamente diseñados
para dar vida a ideas, estilos y conceptos visuales únicos. Cada imagen representa un proceso de exploración
creativa, donde la imaginación se transforma en resultados visuales a través de modelos de inteligencia artificial.

El objetivo de este proyecto es desarrollar habilidades en el uso de herramientas de IA generativa como apoyo en
el diseño visual, la creatividad digital y la experimentación artística, además de construir un portafolio que refleje
mi evolución en este campo.
